\# QuickBite Food Delivery Backend



A food delivery backend application built using Spring Boot, Spring Security, JWT Authentication, Spring Data JPA, and MySQL.



\## Tech Stack



\- Java 17

\- Spring Boot

\- Spring Security

\- JWT Authentication

\- Spring Data JPA

\- MySQL

\- Maven



\## Features



\- User Authentication \& Authorization

\- Restaurant Management

\- Category Management

\- Food Item Management

\- Cart Management

\- Order Management

\- Global Exception Handling



\## Roles



\- ADMIN

\- CUSTOMER

\- RESTAURANT\_OWNER

\- DELIVERY\_PARTNER



\## Setup



1\. Clone the repository

2\. Rename `application-example.properties` to `application.properties`

3\. Update database credentials and JWT secret

4\. Run the application
